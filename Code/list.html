Tag HTML

  1. html -> head -> body
  2. didalam tag head ada title
  3. tag p : digunakan untuk membuat paragraf
  4. color : memberikan warna
  5. font-size : mengubah ukuran font/tulisan
  6. font family mengubah jenis font/tulisan
  7. tag igm src=""> untuk menambahkan gambar
  8. src => source/sumber dimana meletakkan nama file gambar, jangan lupa sertakan ekstensinya.jpg /.jpg/.gif
  9. widht: mengatur lebar -> pake % atau px
  10. height : mengatur tinggi => pake % atau px
  11. border : 10px solid pink
  12. - 10 px : ketebalan dari border
  13. - solid : jenis dari border bisa berupa dotted/dashed
  14. - pink : warna
  15. border-radius : 90% => melengkungkan gambar
  16. tag a href ="" => untuk link
